# Webovky 1 - Mňau Café

**Kočičí kavárna**  
Webová aplikace pro fiktivní kočičí kavárnu v centru města. Hosté si mohou prohlédnout menu, seznámit se s kočičími rezidenty, přidat položky do košíku a odeslat kontaktní zprávu.

## Technologie
- Frontend: React + React Router
- Stylování: Bootstrap 5 + vlastní CSS
- Správa stavu: Context API + localStorage
- Hosting: Vercel: [Mňau Café](https://kotocafe-mnau.vercel.app/)

## Funkce
- 🐾 Představení kočičích rezidentů kavárny
- ☕ Katalog nápojů a jídla s filtrováním podle kategorií
- 🛒 Košík s perzistencí přes localStorage
- 👤 Registrace a přihlášení uživatele
- 📬 Kontaktní formulář
- 📱 Responzivní design
